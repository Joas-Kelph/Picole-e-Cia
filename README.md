# Picolé & Cia

Projeto desenvolvido como atividade avaliativa do SENAI, com foco em desenvolvimento web utilizando HTML, CSS e JavaScript.

## Sobre o projeto

O objetivo deste projeto foi criar uma página temática de picolés, aplicando conceitos de:

- Estruturação HTML
- Estilização com CSS
- Interatividade com JavaScript
- Responsividade
- Organização visual de interface

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Instituição

Projeto desenvolvido para avaliação prática do SENAI.

## Prévia

Interface web temática inspirada em uma loja de picolés, com design moderno e responsivo.
